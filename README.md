# manti_finance_dev

> Personal Finance Copilot -- piattaforma di gestione patrimoniale personale

## Stack
- Frontend: React + TypeScript
- Backend: Firebase (Firestore, Auth, Storage, Functions)
- CI/CD: GitHub Actions

## Moduli
- Investimenti & PAC
- Payroll Engine
- Mutuo & Ammortamento
- Previdenza & TFR
- Decision Engine
- Document Intake
- Cash Flow & Conti
- Kindergarten (area separata)
- Dashboard centrale
- Snapshot & Audit Trail

## Sviluppo
Flusso: issue → branch → PR → review → merge

## Regole
- Single source of truth
- Document-first
- Low-cost by design
- Audit trail su ogni modifica
- Kindergarten separato dal patrimonio personale
